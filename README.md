### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Bope142/secure-password-generator.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd secure-password-generator
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```
